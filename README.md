# Spotify-challenge

There are three page in this app. 

First page(Search) : http://127.0.0.1:8080/ is redirected to http://127.0.0.1:8080/tracks automatically

Here genre of music is taken and transmitted.

Second page(Result) : Here is list of all ten tracks of given genre. You can go to first page with button.

Third page(Not found): If there is not any data about this genre, it is redirected to first page using button.





# Libraries:

Requests: For request and getting response from Spotify API.

Base64: For authentication in Spotify API.

Json: For convert values from json to Python values.

Random: For choose random singer.

Os and Sys: For giving path of genre.json
